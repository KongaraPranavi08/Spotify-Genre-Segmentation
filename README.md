# Spotify Songs Genre Segmentation

## Project Overview
This project uses Machine Learning to group Spotify songs into clusters based on their audio features and recommends similar songs using K-Means Clustering.

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Matrix
- Feature Scaling
- K-Means Clustering
- PCA Visualization
- Music Recommendation System

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Dataset
Spotify Songs Dataset

## Project Workflow
1. Load the dataset
2. Preprocess the data
3. Perform EDA
4. Create correlation matrix
5. Scale features
6. Apply K-Means clustering
7. Visualize clusters using PCA
8. Build a song recommendation system

## Output
- Clustered Spotify songs
- Visualizations and plots
- Song recommendations based on similar clusters
